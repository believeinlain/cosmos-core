Detailed Description

The COSMOS API libraries contain all the functions necessary to support COSMOS. They are broadly divide into general support libraries and specific device libraries. The support libraries include:

  • Agent Server and Client Library
  • Coordinate conversion library
  • Data Management
  • DEM Library
  • Elapsed Time
  • Ephemeris library
  • Earth Geomagnetic Field calculation
  • JPL Ephemeris Library
  • JSON Library
  • Matrix and vector math library
  • NRLMSISE-00 Library
  • Physics Simulation Library
  • COSMOS SLIP encoding library
  • Socket Library
  • String handling library
  • Time handling library
  • File Transfer Library
